GAMMA Modular SAR Processor (MSP) Documentation

MSP Users Guide

The Users Guide gives an introduction on the Gamma Modular SAR Processor (MSP). It tells the user the functionality of the software and in what order the programs are typically used to process SAR raw.

MSP Reference Manual

The Reference Manual describes the individual programs of the Gamma Modular SAR Processor (MSP). It tells the user what a certain program does and how it is used.

MSP Installation

The Installation Manual provides information on how to install the software. If you have the source code distribution, there is information on how to compile and link the source code.
