DIFF&GEO display tools

INTRODUCTION
There are two main approaches to display the various image data:
1. To use one of the display programs (names start with dis).
2. To first use one of the programs to generate a SUN raster or BMF format files (names start with ras), and then to display the output images with an external viewer such as xv.

(Notice that the SUN raster or BMP format files may also be displayed with the program disras with several useful features such as scroll bars, a zoom window, and the display of the location of the cursor together with the local data value.)

Special DIFF&GEO display tools

disdem_par Display DEM as shaded relief. Supported formats: EQA, UTM, OMCH, TM, LCC, PS, PC, AEAC, LCC2, OM, HOM. For the cursor location the map coordinates and the height are displayed.
disras_dem_par Display SUN or BMP rasterfile in map geometry. For the cursor location the map coordinates and the height are displayed.
gcp_2ras Extract corresponding points in two SUN raster or BMP format images. The images are displayed side by side and the cooresponding points are selected using the mouse. The point coordinates in two images are written to a text format file.

Special DIFF&GEO tools to generate SUN/BMP format images

rasshd DEM as shaded relief (float)
