GAMMA Interferometric Point Target Analysis Software (IPTA): Reference Manual

dist_pt

ANSI-C programs: dist_pt.c

NAME
dist_pt - generate point displacement histories in text format

SYNOPSIS
dist_pt <plist> <pmask> <pdata> <pSLC_par> <np_ref> <dx> <dy> [output]

<plist> (input) point list (int)
<pmask> (input) point data stack of mask values ( set to - to accept all points)
<data>
 (input) point data stack (float data)
<np_ref> reference point number, (beginning with 0)
<dx> across pixel spacing (m)
<dy> down pixel spacing (m)
[output]  (output) text file, enter - for output to stdout

EXAMPLE

dist_pt pt_utm.coord  pmsk2  pdh_err 141567 20. 20. dh_err.dat

Generates a text format plot file containing the distance from the reference point for each point in the list and the associated value in the input data file pdh_err

INSTALLATION
Source code dist_pt.c in ./IPTA/src, executable version dist_pt in ./IPTA/bin.

DESCRIPTION
The program dist_pt is used to generate a text format file to plot the data values as a fucniton of distance from the reference point using the xmgrace plotting program. The output file has a format as shown in Table 1. with column 1 the distance from the reference point in meters, column 2 is the value from the data file, column 3 is the point number index, column4 the layer number, column 5 the x coordinate of the point and column 6 is the y coordinate of the point from the point list.

The command to plot using xmgrace is as follows:

xmgrace dh_err.dat

Each point in the data set will be plotted as a point in the xmgrace plot window.

OPTIONS
The user can specify a text list of point numbers to generate output displacement time series. If  a - is entered for the plist1_txt file, then all points not masked out by pmask will be processed.
