GAMMA Differential Interferometry and Geocoding Software (DIFF/GEO): Documentation

DIFF Users Guide

The Users Guide gives an introduction on the GAMMA Differential Interferometry and Geocoding Software (DIFF&GEO). It tells the user the functionality of the software and provides an overview over the sequence used in the processing of SAR differential interferometric products.

DIFF Reference Manual

The Reference Manual describes the individual programs of the GAMMA Differential Interferometry and Geocoding Software (DIFF&GEO). It tells the user what a certain program does and how it is used.

DIFF Installation

The Installation Guide provides software installation information.
