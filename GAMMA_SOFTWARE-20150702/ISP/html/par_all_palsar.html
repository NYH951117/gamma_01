Gamma ISP: Reference Manual

Programs for generation of SLC parameter files and image files for ALOS PALSAR SLC data

ANSI-C programs: par_EORC_PALSAR.c, par_ERSDAC_PALSAR.c.



For par_EORC_PALSAR

NAME
par_EORC_PALSAR - supports SLC Level 1.1 SLC products produced by EORC/JAXA in CEOS format. This program generates the ISP image parameter file from the CEOS meta data and reformats the SLC data records by removing the 412 byte line header in order to make the SLC data compatible with the GAMMA Software processing.

SYNOPSIS

par_EORC_PALSAR <CEOS_leader> <SLC_par> <CEOS_data> [SLC]

<CEOS_leader> (input) CEOS leader file for PALSAR Level 1.1 SLC data (LED...)
<SLC_par>
 (output) ISP SLC parameter file (example: <date>.slc.par)
<CEOS_data>  (input)  PALSAR CEOS format Level 1.1 SLC (IMG...)
<SLC>
 (output) reformated PALSAR SLC (enter - for none, example: <date>.slc)



For par_ERSDAC_PALSAR

NAME
par_ERSDAC_PALSAR - supports SLC Level 1.1 SLC products produced by ERSDAC in non-CEOS, VEXCEL format. This program generates the ISP image parameter file from the PASLL*.SLC.par meta data. The SLC data have no header and are big-endian float complex (4 bytes real, 4 bytes imaginary/sample).

SYNOPSIS

par_ERSDAC_PALSAR <ERSDAC_SLC_par> <SLC_par>

<ERSDAC_SLC_par> (input) ERSDAC SLC parameter file for PALSAR Level 1.1 SLC data (PASL11*.SLC.par)
<SLC_par>
 (output) ISP SLC parameter file (example: <date>.slc.par)
