Program for generation of SLC parameter file and image file for SIR-C SLC data

ANSI-C programs: par_SIRC.c.


NAME
par_SIRC - reads the LEADER file in CEOS format and generates the ISP SLC parameter file. The user can specify the state vectors time reference.

SYNOPSIS

par_SIRC <CEOS_leader> <SLC_par> <UTC/MET>


<CEOS_leader> (input) CEOS leader file provided by processing facility with SLC data
<SLC_par> (output) ISP SLC parameter file
<UTC/MET> state vectors time reference, MET or UTC (default)
