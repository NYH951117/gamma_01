GAMMA Land Application Tools (LAT): Documentation

LAT Users Guide

The Users Guide gives an introduction on the GAMMA Land Application Tools (LAT). It tells the user the software functionalityand provides support in the development of land applications based on SAR and SAR interferometry.

LAT Reference Manual

The Reference Manual describes the individual programs of the Gamma Interferometric SAR Processor (ISP). It tells the user what each program does and how it is used.

LAT Installation

The Installation Manual provides software installation information.
