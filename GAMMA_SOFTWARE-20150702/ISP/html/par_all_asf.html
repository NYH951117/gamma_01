Gamma ISP: Reference Manual

Programs for generation of SLC parameter files and image files for images generated by ASF

ANSI-C programs: par_ASF_91.c, par_ASF_96.c, par_ASF_PRI.c, par_ASF_RSAT_SS.c.



For par_ASF_91

NAME
par_ASF_91 - reads the LEADER and the TRAILER files and generates the ISP SLC parameter file.

SYNOPSIS

par_ASF_91 <CEOS_leader> <CEOS_trailer> <SLC_par>

<CEOS_leader> (input) CEOS leader file provided with the data
<CEOS_trailer>
 (input) CEOS trailer file provided with the data
<SLC_par>
 <output> ISP SLC parameter file (example yyyymmdd.mli)



For par_ASF_96

NAME
par_ASF_96 - reads the LEADER file and generates the ISP SLC parameter file.

SYNOPSIS

par_ASF_96 <CEOS_leader> <SLC_par>

<CEOS_leader> (input) CEOS leader file provided with the data
<SLC_par>
 <output> ISP SLC parameter file (example yyyymmdd.mli)



For par_ASF_PRI

NAME
par_ASF_PRI - reads the LEADER file and the image data file, and generates the ISP SLC parameter file, reformatting the image data file.

SYNOPSIS

par_ASF_PRI <CEOS_leader> <CEOS_data> <GRD_par> <GRD>

<CEOS_leader> (input) CEOS leader file provided with the data
<CEOS_data>
 (input) CEOS ground range detected image data from ERS, Radarsat, or JERS (binary)
<GRD_par>  (output) ISP ground range image parameter file (example yyyymmdd.mli.par)
<GRD>
 <output> ISP ground range image (example yyyymmdd.mli) (enter -  for none, short integer)



For par_ASF_RSAT_SS

NAME
par_ASF_RSAT_SS - reads the LEADER file and the image data file, and generates the ISP SLC parameter file, reformatting the image data file.

SYNOPSIS

par_ASF_RSAT_SS <CEOS_leader> <CEOS_data> <GRD_par> <GRD>

<CEOS_leader> (input) CEOS leader file provided with the data (RADARSAT-1 ScanSAR)
<CEOS_data>
 (input) CEOS ground range detected image data (RADARSAT-1 ScanSAR)
<GRD_par>  (output) ISP ground range image parameter file (example yyyymmdd.mli.par)
<GRD>
 <output> ISP ground range image (example yyyymmdd.mli) (enter -  for none, short integer)
