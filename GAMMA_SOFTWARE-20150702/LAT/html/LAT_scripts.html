LAT Scripts

tcsh-scripts: LAT/scripts

PURPOSE
A number of TC-shell (tcsh) scripts are included to support the efficient use of the LAT programs, primarily for polarimetric processing. The scripts allow to run often used processing sequences, more automatically, (guided by the script).
The scripts may also be taken as examples on how to write new scripts, matching the exact user demands.

NAMES AND FUNCTIONS

Polarimetric tools:

CLOUDE_DEC Eigenvector-based target decomposition (Cloude, 1996)
COH_MATRIX Generation of elements of coherency matrix T3
COV_MATRIX Generation of elements of covariance matrix C3
FD3C_DEC Surfacem double-bounce, volume scattering decomposition, (Freeman, 1998)
HUYNEN_DEC Equivalent single target decomposition, Huynen decomposition
KENNAUGH_MATRIX Generation of elements of Kennaugh matrix K4

DIAGNOSTICS
Using the scripts without arguments returns information on the input parameter list.

If a script does not result in the expected result (i.e. the processing stops, the resulting file just contains NULL values, etc.) we recommend to execute the program of the script individually and to test for each program the input and output for possible indications of the source of an error.
