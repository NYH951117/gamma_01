res_map

ANSI-C program: res_map.c

NAME
res_map - Uses the interferometrically derived terrain heights (in slant range geometry) and true ground ranges (in slant range geometry) to derive a rectified height map.

SYNOPSIS
res_map <hgt> <gr> <data> <SLC_par> <OFF_par> <res_hgt> <res_data> [nr] [naz] [azps_res] [loff] [nlines] <report_file>
 

<hgt> (input) height file in slant range geometry
<gr> (input) ground range file in slant range geometry
<data> (input) data file in slant range geometry (float) (intensity *.pwr or correlation *.cc)
<SLC_par> (input) ISP parameter file of reference SLC
<OFF_par/par> (input) offset/interferogram processing parameters
<res_hgt> (output) resampled height file in ground range geometry
<res_data> (output) resampled data file in ground range geometry
[nr] range samples for L.S. estimate (default = 7, must be odd)
[naz] azimuth samples for L.S. estimate (default = 7, must be odd)
[azps_res] azimuth output map sample spacing in meters (default=azimuth spacing)
[lof] offset to starting line to begin height calculations (default = 0)
[nlines] number of lines to calculate (default=0, to end of file)

EXAMPLE
res_map 1352_1610.hgt 1352_1610.gr 1352_1610.pwr1 1352.par 1352_1610.off 1352_1610.rhgt 1352_1610.rpwr1 9 9

DESCRIPTION
Uses the interferometrically derived terrain heights (in slant range geometry) and true ground ranges (in slant range geometry) to derive a rectified height map in TCN coordinates, with the cross-track (C) coordinate following the WGS-84 ellipsoid. The rectification of the height data requires resampling in both azimuth and cross-track directions.

One of the backscatter intensities or the interferometric correlation may be projected into the TCN coordinates and combined with the height information. The resulting image (containing the terrain height and the SAR image intensity) is written into the output file. The display and generation of hard copies of the resampled height map (in rectified coordinates) together with the image intensities is supported by a number of display programs.
