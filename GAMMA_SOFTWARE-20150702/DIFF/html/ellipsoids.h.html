ellipsoids.h

List of Ellipsoid Parameter Sets.

To display the current list contained in ellipsoids.h press here.

Used by the programs create_dem_par, coord_trans to support the Ellipsoid Parameter definition.

Each line of the list contains the following parameters (corresponding to the structure ELLIPS defined in typedef_DIFF.h:

Structure Elements Example Definition
name[80] Airy 1830 name of datum
ellips_num 1 ellipsoid number (line in the list)
ra 6377563.396 semi-major axis [m]
rec_f 299.3249646 reciprocal of flattening (1/f)
