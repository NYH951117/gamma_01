ORRM orbit state vector file: <orbit>.orrm

(ASCII-file)

NAME
<orbit>.orrm - contains orbit state vector data from ESA, but is now obsolete. The file contains the state vectors and velocities for the sensor (ERS-1). The state vectors and velocities are used to estimate the interferometric baseline. The precision orbit product *.PRC has replaced this older format for state vectors.

SYNOPSIS
Non-executable and obsolete.

EXAMPLE

ORRM_911021MCCF0183.E102:18:27
0 0 15266 0 15266 86340000 1991/10/21 0:51: 7
15266 0 1346 1669.252283 -1781.4552 6718.5946 -6.8175479 2.2782278 2.2926923 0
15266 60000 1346 1257.8307 -1639.5883 6842.8982 -6.8918049 2.4492421 1.8494171 0
15266 120000 1346 842.8012 -1487.7264 6940.3805 -6.9377854 2.6112437 1.3989348 0
15266 180000 1346 425.8638 -1326.4368 7010.6614 -6.9553996 2.7633585 0.9430021 0

(Example represents ORRM orbit state vector file: *.orrm for orbit 1352. In addition to the orbit data for orbit 1352 it contains data for further orbits, starting with orbit 1346).

FORMATS AND DEFINITIONS
The first two lines contain auxiliary data. The format and parameters for line 3 and continuation are as follows (line format is defined in the ISP type definition file typedef_ISP.h:

3. line and following (int mjd, utc, orb_num, double x, y, z, vx, vy, vz, int TAI_UTC_delta):
mdj: modified Julian date, which is (julian date-2433282.5) e.g.: 1.1.1990 = 14610
utc: time (in msec) since midnight UTC
orb_num: orbit number
x, y, z: sensor position (in km) in Conventional Terrestrial System
vx, vy, vz: sensor velocity (in km/sec) in Conventional Terrestrial System
TAI_UTC_delta: difference (in sec) between the International Atomic Time and UTC

DESCRIPTION
<orbit>.orrm - contains ORRM orbit data as provided by ESA. The ORRM orbit state vector file contains the state vectors and velocities for the sensor (ERS-1). The state vectors and velocities are used to estimate the interferometric baseline. The program state_vec interprets the ORRM orbit state vector file and writes state vectors to the SLC parameter file. This format is now obsolete and no longer supported by ESA.

OPTIONS
The ORRM orbit state vector file <orbit>.orrm is not required if the state vectors from the SLC leader/header files are available.
