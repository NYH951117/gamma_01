OPOD_vec

ANSI-C program: OPOD_vec.c

NAME
OPOD_vec  - extract state vectors from OPOD precision orbit state vector files and copy to the ISP SLC/MLI image parameter file

SYNOPSIS
OPOD_vec <SLC_par> <EOF> [nstate]

<SLC_par> (input/output) ISP SLC/MLI  image parameter file
<DOR>  (input) (input) Sentinel-1 OPOD orbit data file (AUX_POEORB or AUX_RESORB)
https://qc.sentinel1.eo.esa.int/aux_resorb/
[nstate] number of state vectors to extract (image +- 35 seconds )

EXAMPLE
OPOD_vec 22331.par S1A_OPER_AUX_RESORB_OPOD_20150402T162033_V20150402T122229_20150402T154000.EOF

DESCRIPTION
OPOD_vec reads state vectors distributed by the Sentinel-1 payload data ground segment. The state vectors are written into the ISP SLC/MLI image parameter file along with the start time and time interval between state vectors.
https://qc.sentinel1.eo.esa.int/aux_poeorb/
https://qc.sentinel1.eo.esa.int/aux_resorb/
