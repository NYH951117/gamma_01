ANSI-C program: create_sar_par.c

NAME
create_sar_par - Create and edit MSP SAR sensor parameter files.

SYNOPSIS

create_sar_par <sar_par>

<sar_par> SAR sensor parameter file


EXAMPLE
create_sar_par SAR.par

DESCRIPTION
create_sar_par creates or edits an existing SAR Sensor Parameter file. The Sensor Parameter file describes the raw data format, radar chirp and analog to digital converter (ADC) characteristics, antenna radiometric calibration file, and the nominal antenna pointing angles.

OPTIONS
For the SAR sensor on the ERS1/2 and JERS-1 satellites the existing SAR sensor parameter files in the ./sensors directory are recommended for use. Processing of RadarSat-1 and SIR-C uses a SAR sensor parameter file created by the programs RSAT_raw and SIRC_proc. Due to the many data acquisition modes of these radars the sensor parameter file is generated from parameters in the CEOS leader file.
