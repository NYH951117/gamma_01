SLC_tab

The SLC_tab is a text file contains a list of the SLC images used for processing and the associated ISP image parameter files. There are two columns in the file with the first column containing the SLC file name including the absolute or relative path.

The second column is the SLC image parameter file (ISP format) and also includes the relative or absolute path. Any line beginning with a # is ignored when the file is read. If a relative path is used, then make sure to use the SLC_tab only in directories where the path is sufficient.

EXAMPLE:

rslc/19960421.rslc     rslc/19960421.rslc.par
rslc/19960422.rslc     rslc/19960422.rslc.par
