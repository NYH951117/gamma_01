GAMMA Land Application Tools (LAT): Reference Manual

Programs for Filtering and Parameter Estimation

ave_cpx Average a set of images containing float complex (fcomplex) data
ave2pwr Average 2 registered backscatter intensity images
average_filter Running average filter
cpxlks Multi-looking for complex valued image (real part: 32-bit float, imaginary part: 32-bit float)
cc_ad Adaptive interferometric coherence estimation
enh_lee Enhanced Lee spatial filter (Lopes et al., 1990)
frost Minimum mean square error filter (Frost et al., 1982)
gamma_map Enhanced Kuan spatial filter (Lopes et al., 1990)
lee Lee spatial filter (Lee et al., 1980)
median_filter Median spatial filter
multi_stat
 Sort MLI image values in a stack and support selection of output image based on average, rank, or percentile
mt_lee_filt
 Multi-temporal Lee directional adaptive filter for float data
mt_lee_filt_cpx Multi-temporal Lee directional adaptive filter for fcomplex data
ratio Estimation of ratio between of two backscatter images
reallks Multi-looking for float data images
sigma2gamma Compute backscatter coefficient gamma0 = (sigma0)/cos(inc angle)
texture Estimation of spatial image texture
temp_filt Filtering of multi-temporal co-registered SAR images
temp_filt_ad Filtering of multi-temporal co-registered SAR images using adaptive spatial mean estimates
temp_lin_var Temporal mean and variability of multi-temporal co-registered SAR images (using linear scale)
temp_log_var Temporal mean and variability of multi-temporal co-registered SAR images (using dB values)

Programs for Test-Area Analysis

histogram Calculation of histograms for polygon regions for file of floats
histogram_ras Calculation of histograms for polygon regions of SUN raster of BMP format images
looks Equivalent number of looks (ENL) estimation for polygon regions
polyras Selection of polygon regions (as part of DISP package)
polyx Extraction of values from data file (float) for polygon regions
polyx_phase Extraction of phase and backscatter intensities for polygon regions
takecut Extraction of float values from a line or vertical cut of a float data file and store in text format for plotting.
takethat Extraction of values along profiles, in polygon regions, or for indicated positions (pixel numbers indicated)
takethat_dem_par Extraction of values along profiles, in polygon regions, or for indicated positions (map coordinates indicated)

Classification Tools

cc_monitoring Classification based on multiple input files using single threshold
mask_class Set values for a class (complement to a class) to a user defined value
multi_class_mapping Classification based on multiple input files using threshold table
ras_majority Majority filtering of 8-bit SUN raster or BMP format classification maps
single_class_mapping Mapping of a single class based on multiple input files using thresholds
validate Validation of classification map using a reference map containing data on the true class

Mathematical and Auxiliary Tools

float2short Convert file of type float (32-bit) to file of type short integer (16-bit)
float2uchar Convert file of type float (32-bit) to file of type unsigned char (8-bit)
lin_comb Calculate linear combination of N float format data files
lin_comb_cpx Calculate linear combination of N fcomplex format data files
lin_comb_ref
 Calculate linear combination of 2 images (float) with option to add or subtract the scaled average of a specified reference region
line_interp Simple interpolator to fill data gaps using a 1-D interpolator
linear_to_dB Conversion of data between linear and dB scaling
mask_op Perform logical operations on mask files (Sun raster or BMP format)
poly_math Perform arithmetic on a polygonal image regions. Applying a unique offset and scale factor for each each region is supported. This program can also be used to extract polygonal regions and copy them to a file with background value of 0.0.
product Calculation of product of two real valued images
product_cpx Calculation of product of two complex valued images
replace_values Replace value(s) of input data file with indicated value or values extracted from a second input data file (float)
restore_float Interpolator to fill image gaps by linear interpolation in 2-D
short2float Convert file of type short integer to file of type float
swap_bytes Byte order swapping between BIG_ENDIAN and LITTLE_ENDIAN (as part of DISP package)
uchar2float Convert file of type unsigned char to file of type float
trigo Compute trigonometric functions of an input file of type float
unw_to_cpx Convert unwrapped phase to an fcomplex interferogram

Visualization Tools

comb_hsi Combine color (HUE, SAT) of the 1. SUN raster or BMP format images with INTENSITY of the 2. SUN raster or BMP format image
drawthat draw an arc, polygon, cross, or point on a Sun raster file at specified positions
frame Set values along image borders to a NULL value
hsi_color_scale Generate color scale for hue/saturation/intensity (HSI) image
ras_ras Multi-looking and left/right flipping of SUN raster and BMP format images
ras_to_hsi Combine 3 SUN raster or BMP format images into a hue/saturation/intensity (HSI) composite
ras_to_rgb Combine 3 SUN raster or BMP format images into a red/green/blue (RGB) composite

Retrieval Algorithms

soil_moisture Soil moisture retrieval from ERS data (Wegmüller et al., 1997)

Polarimetry Tools

diplane_helix Calculate diplane and helix component from the right and left circularly polarized components
haalpha Calculate H, anisotropy, alpha polarimetric decomposition (Cloude-Pottier)
m-alpha Calculate m-alpha decomposition derived using the Stokes parameters. m is the degree of polarization and alpha is the scattering parameter in the H/A/alpha decomposition
m-chi Calculate m-chi polarimetric decomposition derived from Stokes parameters.  m is the degree of polarization and 2*chi is the latitude of the Stokes vector on the Poincaré sphere
m-delta Calculate m-delta polarimetric decomposition derived from Stokes parameters. m is the degree of polarization and delta is the H-V phase difference
pauli Calculate Pauli polarimetric decomposition from HH, VV, and HV SLC images
polcoh Calculate coherence matrix T elements from Pauli decomposition alpha, beta, gamma
polcovar Calculate covariance matrix C elements from HH, VV, and HV SLCs
quad2cp Synthesize Compact Polarization Data from quad-pol HH, VV, HV, VH SLC data
stokes Calculate Stokes parameters from SLC images with H and V receive polarization and the same transmit polarization
stokes_qm Calculate quantitative measures from Stokes parameters (e.g. degree of polarization m, H-V phase difference delta, H/anisotropy/alpha, sin(2*chi))
wolf Calculate Wolf coherence matrix from H and V polarized SLC images

LAT Scripts

CLOUDE_DEC eigenvector-based target decomposition (Cloude, 1996)
COH_MATRIX Generation of elements of coherency matrix T3
COV_MATRIX Generation of elements of covariance matrix C3
FD3C_DEC Surface, double-bounce, volume scattering decomposition, (Freeman, 1998)
HUYNEN_DEC Equivalent single target decomposition, Huynen decomposition
KENNAUGH_MATRIX Generation of elements of Kennaugh matrix K4
