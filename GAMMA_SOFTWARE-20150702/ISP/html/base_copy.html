Gamma ISP: Reference Manual

base_copy

ANSI-C program: base_copy.c

NAME
base_copy - Determine baseline file for SLC sub-section (extracted using SLC_copy).

SYNOPSIS
base_copy <SLC-1_par> <baseline_1> <SLC-2_par> <baseline_2> [time_rev]
 

<SLC-1_par> (input) ISP image parameter file of the reference SLC
<baseline_1> (input) baseline file derived using the reference SLC geometry
<SLC-2_par> (input) ISP image parameter file corresponding to the subsecton of the reference SLC
<baseline_2> (output) baseline file derived using the geometry and timing of the SLC subsection
[time_rev]; SLC image normal=1, time-reversed = -1 (default=1)

EXAMPLE
base_copy 05549.slc_esa.par 05549_11270.base 05549.rslc_esa.par 05549_11270.base_res

DESCRIPTION
base_copy is used to copy the baseline file after SLC sub-section extraction (with the ISP program SLC_copy). The baseline is adjusted according to the geometric parameters in the reference SLC parameter file of the sub-section SLC. The baseline is defined as the baseline at the first line of the reference SLC. When extracting a sub-section of the SLC this reference position changes and therefore the baseline must be calculated for the segment. The baseline component rates from the original baseline file are used to calculate the baseline values for the center of the interferogram created using the SLC subsection as the reference image.

In differential SAR interferoemtry base_copy is required when estimating the perpendicular component of the baseline based on the fringe frequency of a large area (possibly excluding the area of surface motion) and applying it to the smaller extracted sub-section with ground surface motion.
