base_perp

ANSI-C program: base_perp.c

NAME
base_perp - Computation of baseline components normal and parallel to look vector.

SYNOPSIS
base_perp <baseline> <SLC-1_par> <interf_par> [right/left] [time_reverse]

<baseline> baseline file
<SLC-1_par> parameter file of first (master) SLC image
<interf_par> interferogram/offset processing parameter file
[right/left] right/left side illumination of SAR
(default: 1=right, -1=left)
[time_reverse] SLC image time reverse flag
(default: 1=normal, -1=reversed)

EXAMPLE
base_perp 1352_1610.base 1352.part 1352_1610.off 1 1

INSTALLATION
Source code base_perp.c in ./src. For compilation adjust and use Makefile: Executable version base_perp in ../bin

AVAILABILITY
Uses ISP type definition file typedef_ISP.h.

DESCRIPTION
base_perp reads the input baseline file and calculates for a number of locations in the image (starting pixel etc defined in interferoemtric processing/offset parameter file) the baseline components parallel and perpendicular to the local look vector.

If the baseline file contains a precision estimate (i.e. if the third and fourth line contains non-zero values) the precision baseline estimate is used, otherwise the initial estimate.

The image locations (range, azimuth pixel numbers) and the baseline components are printed to standard output.

OPTIONS
None.

SEE ALSO
typedef_ISP.h, baseline file.
