MLI_cat

ANSI-C program: MLI_cat.c

NAME
MLI_cat - Concatenate a pair of MLI images using bicubic spline interpolation

SYNOPSIS
MLI_cat <MLI-1> <MLI-2> <MLI1_par> <MLI2_par> <OFF_par> <MLI-3> <MLI3_par>

<MLI-1> (input) MLI-1 image single look float
<MLI-2> (input) MLI-2 image to be appended to MLI-1
<MLI1_par> (input) MLI-1 ISP image parameter file
<MLI2_par> (input) MLI-2 ISP image parameter file
<OFF_par> (input) ISP offset parameter file containing offset polynomials between MLI-1 and MLI-2
<MLI-3>
 (output) concatenated MLI
<MLI3_par> (output) ISP image parameter file for concatenated image

EXAMPLES

MLI_cat mli/20110324T123632.mli mli/20110324T123637.mli mli/20110324T123632.mli.par mli/20110324T123637.mli.par cat_mli/20110324T123632_20110324T123637.off cat_mli/20110324T123632.slc cat_mli/20110324T123632.mli.par

Concatenate 2 MLI images and generate an output MLI. Use the SLC offset polynomial information in the OFF_par file to concatenate the files. MLI images must be single-look 

DESCRIPTION

MLI_cat is a program to concatenate 2 single-look MLI images. The offset parameter file OFF_par is used to contain information on spatial offset between the MLI images  in both range and azimuth.  A bicubic-spline interpolation is used to append a resampled version of MLI-2 on to the end of MLI-1.  The offset polynomial information is obtained from the SLC offset measurements.  See the documentation for SLC_cat that describes the process for obtaining the offset polynomials in range and azimuth.  This program will be used primarily to concatenate MLI images from interferograms
