GAMMA Interferometric Point Target Analysis (IPTA) Documentation

IPTA Users Guide

The Users Guide gives an introduction on the GAMMA Interferometric Point Target Analysis Software (IPTA). It tells the user the functionality of the software and provides an overview over the sequence used in typical processing tasks.

IPTA Reference Manual

The Reference Manual describes the individual programs of the GAMMA Interferometric Point Target Analysis Software (IPTA). It tells the user what a certain program does and how it is used.

IPTA Installation

The Installation Manual provides information on how to install the software. If you have the source code distribution, there is information on how to compile and link the source code.
