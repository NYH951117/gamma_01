Reformat EORC processed JERS-1 SLC data and generate the ISP parameter file

ANSI-C programs: par_EORC_JERS_SLC.c

NAME
par_EORC_JERS_SLC - Reformat EORC processed JERS-1 SLC data and generate the ISP parameter file

SYNOPSIS

par_EORC_JERS_SLC <CEOS_SAR_leader> <PROC_par> [

<CEOS_SAR_leader> (input)CEOS SAR leader file for JERS SLC data processed by EORC/NASDA
<SLC_par> (output) ISP SLC image parameter file
[CEOS_data]  (input) CEOS format SLC data (IMOP_01.DAT, enter - for none)
[SLC] (output) reformated JERS SLC (example: yyyymmdd.slc, enter - for none)

EXAMPLES
par_EORC_JERS_SLC SARL_01.DAT 19960101.slc.par DAT_01.001 19960101.slc

Reads CEOS leader file SARL_01.DAT and creates the SLC parameter file 19960101.slc.par. The SLC data are then reformated, scaled, and stored in the  SLC file 19969191.slc.

DESCRIPTION
Reads the CEOS leader file and creates the SLC image parameter file. Then this program reads the SLC image data and reformates the data. The SLC data are stored in the file as groups of 768 range lines. There are a total of 22 patches in the file. These are each read and then written to disk after a corner turn.  A scale factor of 1.e-12  is applied to each complex value to bring the values into the range of unity.
