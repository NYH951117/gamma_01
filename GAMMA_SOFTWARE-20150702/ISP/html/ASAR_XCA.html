Interpretation of ASAR external calibration data file (ASA_XCA)

ANSI-C programs: ASAR_XCA.c

NAME
ASAR_XCA - reads ENVISAT ASAR external calibration data file as provided by ESA, writes calibration factors for the different modes and data products to the screen and generates antenna diagram files in the format used by GAMMA software.

SYNOPSIS

ASAR_XCA <ASAR_XCA> <antenna> [swath] [pol]

<ASA_XCA> (input) ASAR external calibration data file (binary)
<antenna> (output) 1-way antenna gain pattern file or '-' (if not provided)
or 'all' to generate all ASAR antenna diagrams
[swath] ASAR swath (IS1,IS2,...IS7;SS1,SS2,...SS5)
[pol] polarization (HH,VV,HV,VH)

EXAMPLES
ASAR_XCA ASA_XCA_AXVIEC20021217_150852_20020413_000000_20031231_000000 ASAR_IS2_VV_antenna.gain IS2 VV

Reads the ASAR external calibration data file and generates:
ASAR_IS2_VV_antenna.gain - ASAR antenna diagram swath IS2, VV polarization
In addition, calibration scaling factors for all modes/products are written to the screen.

ASAR_XCA ASA_XCA_AXVIEC20021217_150852_20020413_000000_20031231_000000 all

Reads the ASAR external calibration data file and generates ASAR antenna diagrams for all swaths and polarizations. In addition, calibration scaling factors for all modes/products are written to the screen.

DESCRIPTION
Antenna diagrams and calibration scale factors of the different ASAR swaths, polarizations, and products are specified in the ASAR external calibration data file which is provided by ESA as a separate product. ASAR_XCA reads ENVISAT ASAR external calibration data file as provided by ESA, writes calibration factors for the different modes and data products to the screen and generates antenna diagram files in the format used by GAMMA software.

In addition, the reference look angles (i.e. center of antenna diagrams) are specied in the external calibration data file and displayed on the screen by ASAR_XCA.
