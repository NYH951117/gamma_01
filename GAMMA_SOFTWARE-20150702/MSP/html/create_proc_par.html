create_proc_par

(ANSI-C program) create_proc_par.c

NAME
create_proc_par - Supports the creation and editing of MSP processing parameter files.

SYNOPSIS
create_proc_par <SAR_param> <PROC_param>

<SAR_param> MSP SAR sensor Parameter file
<PROC_param> MSP processing parameter file


EXAMPLE
create_proc_par ERS1_ESA.par p8560.slc.par

DESCRIPTION
create_proc_par creates an MSP Processing Parameter file and supports editing of the parameter values for existing files. Generally this program is used only to modify the processing parameters for which specific parameter generation program does not exit. The MSP uses two parameter files for SAR processing. The MSP sensor parameter file describes the raw data format and the SAR system parameters that do not change from pass to pass. The processing parameter file has the scene dependent parameters and values of parameters selected by the user for processing of this particular scene. The sensor parameter file does not change, while the values of the processing parmaeters are scene dependent.
