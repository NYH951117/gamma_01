Gamma ISP: Reference Manual

Program for generation of ISP SLC parameter file for image files obtained with the MSP module

ANSI-C programs: par_MSP.c.


NAME
par_MSP - reads the MSP SLC parameter file and generates the corresponding parameter file for the ISP.

SYNOPSIS

par_MSP <MSP_sar> <MSP_proc> <SLC_par> <image_format>

<SAR_par> (input) MSP SAR sensor parameter file
<PROC_par> (input) MSP processing parameter file
<SLC_par> (output) ISP image parameter file
<image_format> image format flag, e.g. FCOMPLEX, SCOMPLEX or FLOAT
