dem_x_y_z

ANSI-C program: dem_x_y_z.c

NAME
dem_x_y_z - Convert DEM heights and positions to WGS84 X, Y, and Z coordinates written to 3 separate files in float or double precision

SYNOPSIS
dem_x_y_z <DEM_par> <DEM> <DEM_X> <DEM_Y> <DEM_Z> [format_flag]

<DEM_par> (input) DEM parameter file
<DEM> (input) DEM file or constant height value
<DEM_X> (output) X-coord. of WGS84 Cartesian coordinates converted from DEM values in a map projection
<DEM_Y> (output) Y-coord. of WGS84 Cartesian coordinates converted from DEM values in a map projection
<DEM_Z> (output) Z-coord. of WGS84 Cartesian coordinates converted from DEM values in a map projection
[format_flag] output data format:
0: FLOAT (default)
1: DOUBLE

EXAMPLE
dem_x_y_z hector_utm_seg.dem_par hector_utm_seg.dem hector_utm_seg.x hector_utm_seg.y hector_utm_seg.z 1

generates three separate files of 8-byte double data values for X, Y, and Z coordinates.

DESCRIPTION
dem_x_y_z calculates the Cartesian coordinates of each point in a DEM using the parameters in the DEM parameter file and the DEM value. Instead of a DEM file a constant height value can be given.
The output format has the same endianness as the input DEM data and consists of three separate files containing the WGS84 Cartesian coordinates X, Y, and Z, respectively. By default the output files are written as 4-byte float data. Optionally, the data can be written as 8-byte double precision values. Instead of a DEM file a constant height value can be given.
