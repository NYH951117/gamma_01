DESCRIPTION
<orbit>.par - contains parameter values for SLC data used throughout the interferometric data processing. <orbit>.par - is either generated from the SAR SLC leader/header information files (provided by the processing facilities together with the SLC data), par_ESA (for CEOS header format provided by D-PAF or ESRIN), par_ASI (for the header format provided by the Italian PAF), par_ASF (for the header format provided by the Alaska SAR Facility), or par_UK (for the header format provided by the UK-PAF), or it is generated from the MSP processing parameters. The formats used in <orbit>.par are defined in the in ISP type definition file typedef_ISP.h.

OPTIONS
If the required SLC leader/header files are missing or have missing values the SLC parameter file <orbit>.par may be edited manually.

SEE ALSO
typedef_ISP.h, par_ESA, par_ASI, par_ASF, par_UK, par_CCRS.

DIAGNOSTICS
